# E-paper Gadget

## Overview

**EpapGadget** is a self contained solution and development platform for Epaper displays. More specifically the PCB is is based on the ESP32-C3, taylored for the WeAct-Epaper4.2" (https://github.com/WeActStudio/WeActStudio.EpaperModule), and battery powered suitable for wall/door/monitor mounting. In this repo continuous development is done on two firmware solutions; Co-op, SingleUser.
